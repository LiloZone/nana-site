Bolsa com sensor
de luminosidade

{{page.titulo}}

Uma bolsa que quando aberta, acende automaticamente um LED que ajuda a enxergar os objetos dentro dela. Quando fechada, o LED apaga.

O controle do LED é feito através de um sensor de luminosidade programado em Arduino. Aberta, recebe luz, acende. Fechada, cessa a luz, apaga. Este tutorial Insere o elemento sensor para um projeto completo: sensor, processamento e atuação.

Montagem

1. CONEXÃO DO MÓDULO EXTENSOR

Ligue o módulo extensor na Porta GND do módulo Arduino.

{{page.titulo}}

Montagem

2. CONEXÃO DO MÓDULO SENSOR

Ligue um módulo conector em cada extremidade do módulo sensor. Um dos módulos conectores deve ser ligado à Porta A1 do módulo Arduino. O outro módulo conector deve ser ligado ao módulo extensor.

{{page.titulo}}

Montagem

3. CONEXÃO DO MÓDULO LED

Ligue o Sinal (S) do módulo LED (botão fêmea) na Porta 9 do módulo Arduino, utilizando o módulo conector. Ligue o Ground (GND) do módulo LED (botão macho) no módulo extensor, utilizando o módulo conector.

{{page.titulo}}

Montagem

4. CONEXÃO DO POWER BANK

Ligue o cabo USB (micro USB) na Porta micro USB do módulo arduino. Em seguida ligue o cabo USB (USB A) no Power Bank. O módulo LED irá acender. Atente-se para deixar o módulo sensor exposto à luz, sem nenhuma intervenção de sombra.

{{page.titulo}}

Montagem

5. ACIONAMENTO DO CIRCUITO

Ao ficar exposto à luz, o módulo sensor irá acender o LED. Ao fechar, protegido da luz, o módulo sensor irá apagar o LED.

Sugestão de Aplicação

Com o circuito pronto, é possível aplicá-lo à sua bolsa de diversas formas. Sugerimos costurar ou colar alfinetes nos módulos sensor, LED e Arduino para fixá-los na parte interna da bolsa.

Atente-se para que o módulo sensor seja aplicado em uma área da bolsa que fique exposta à luz quando aberta e protegida da luz quando fechada. Para que o LED acenda ao abrir a bolsa e apague ao fechar.
