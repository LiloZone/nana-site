Tutorial

Colar
com LEDs

{{page.titulo}}

Um colar que acende 2 LEDs quando colocado. O próprio fecho funciona como interruptor, fechando o circuito e acendendo os LEDs.

Ilustra como funciona um circuito elétrico e como conectar mais de um atuador na mesma fonte de energia.

{{page.titulo}}

Módulos

1 x módulo bateria

3 x módulo conector G

2 x módulo LED

{{page.titulo}}

Montagem

1. PREPARAÇÃO DOS MÓDULOS

Monte um circuito com todos os módulos indicados, utilizando o módulo bateria na posição central.

{{page.titulo}}

Montagem

2. CONEXÃO DOS MÓDULOS

Conecte um módulo LED em cada extremidade do módulo bateria, e um módulo conector em cada extremidade dos módulos LED.

{{page.titulo}}

Montagem

3. ACENDIMENTO DOS LEDS

Para acender o LED do colar, encaixe os botões das duas pontas dos módulos conectores, fechando o circuito.

{{page.titulo}}

Materiais para aplicação

Tiras de tecido de cores variadas

{{page.titulo}}

Sugestão de Aplicação

Com o circuito pronto, é possível criar diversas estruturas de design para finalizar o seu colar. Sugerimos utilizar técnicas de nós e amarrações com tiras de tecido.

Optamos por criar uma amarração mais fechada no centro possibilitando que o módulo bateria fique escondido, deixando apenas os módulos LEDs visíveis

O acabamento pode ser feito com as próprias tiras de tecido, por meio de nós de finalização.
